# The Green function for the Helmholtz operator

Solving the _Helmholtz equation_ is a standard problem in almost any course in mathematical physics, and a simple way to introduce the notion of _Green functions_. I solve it here in full detail, explaining every step of the solution. Originally written on February 22, 2005.
